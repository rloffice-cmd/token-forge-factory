# 🧠 המוח השני - Telegram Memory Bot

בוט טלגרם אישי שמשמש כזיכרון חכם. שולחים הודעה - הוא זוכר. שואלים שאלה - הוא עונה.

## מה זה עושה?

- **שמירת מידע** - שלח הודעה, הבוט מקטלג אוטומטית (קטגוריה, נושא, תגיות, חשיבות)
- **שליפה בשפה טבעית** - שאל "מה הייתה המסקנה מהפגישה עם דוד?" ותקבל תשובה
- **ניהול משימות** - "תזכיר לי לחזור לדוד" → נשמר כמשימה עם עדיפות
- **העברת הודעות** - העבר מ-WhatsApp, מייל, צ'אטים → הכל נקלט ומקוטלג
- **פרטי** - רק הבעלים יכול לגשת, אף אחד אחר

## הקמה מהירה

### 1. יצירת בוט טלגרם
- שלח `/newbot` ל-@BotFather בטלגרם
- קבל את ה-Bot Token

### 2. קבלת Chat ID
- שלח הודעה ל-@userinfobot בטלגרם
- קבל את ה-Chat ID שלך

### 3. קבלת Gemini API Key (חינם)
- היכנס ל-https://aistudio.google.com/apikey
- צור API Key חדש

### 4. הרצה מקומית
```bash
cp .env.example .env
# ערוך את .env עם הפרטים שלך

npm install
npm run db:init
npm run dev
```

### 5. דיפלוי ל-Railway (אופציונלי)
1. צור חשבון ב-railway.com
2. צור פרויקט חדש → הוסף PostgreSQL
3. חבר את ה-GitHub repo
4. הגדר Environment Variables:
   - `BOT_TOKEN`
   - `OWNER_CHAT_ID`
   - `GEMINI_API_KEY`
   - `DATABASE_URL` (מתמלא אוטומטית מ-Railway)
   - `NODE_ENV=production`

## שימוש יומיומי

```
# שמירת מידע - פשוט שלח הודעה
"הדוזאג' של הריטלין הוא 20mg בבוקר ו-10mg בצהריים"
"המסקנה מהפגישה עם דוד: להתקדם עם הפרויקט עד סוף החודש"
"מחיר הלפטופ ב-KSP: 4,500 ₪"

# שליפת מידע - שאל שאלה
"מה הדוזאג' של הריטלין?"
"מה הייתה המסקנה מהפגישה עם דוד?"
"כמה עלה הלפטופ?"

# משימות
"תזכיר לי לחזור לדוד מחר"
"צריך לקנות חלב"
"משימות" → רשימת משימות
"בוצע 5" → סימון משימה #5

# העברת הודעות
העבר הודעה מ-WhatsApp → נשמר אוטומטית
```

- **Railway** (hosting)
